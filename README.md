# «Сапёр» (англ. Minesweeper) игра-головоломка 🤯

### ⚙️ Установка зависимостей:
```
npm install
```

### 🚀 Запуск проекта с локальным сервером:

```
npm run start
```

### 🛠️ Cборка проекта в папку build :
```
npm run build
```

#### Работающий пример в общем доступе [туть](https://turov.github.io/minesweeper/).
